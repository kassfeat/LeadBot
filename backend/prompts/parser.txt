Ты получишь полный транскрипт диалога между ботом маркетингового агентства и клиентом.

Извлеки из диалога факты о клиенте и его задаче и верни строго валидный JSON со следующими полями (если поля нет — опусти его, не выдумывай):

{
  "name": "имя клиента",
  "contact": "телефон / email / @telegram",
  "company": "название компании / проекта",
  "niche": "сфера деятельности",
  "task": "что хочет клиент, суть задачи",
  "budget": "бюджет",
  "deadline": "сроки",
  "extra": "всё остальное важное одной строкой"
}

Возвращай только JSON, без обрамляющего текста и без markdown-блоков.
